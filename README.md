# This is a template!

I made this so I don't have to make a new file setup every time my online course [The Odin Project](https://theodinproject.com)

## How to Use

### 1. Clone the repository

```bash
git clone https://github.com/your-username/your-repo.git
```

### 2. Install dependencies

```bash
cd your-repo
npm install
```

### 3. Run the project

```bash
npm start
```

### 4. Open in browser

```bash
Go to [http://localhost:3000](http://localhost:3000)
```
